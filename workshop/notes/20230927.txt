스프링 부트 의존성
    VS Code에서 의존성 추가하는 방법
        [1] Ctrl + Shift + P 로 커맨드창을 염
        [2] "Spring Initiallizer : Adding Startars..." 를 눌러서 의존성을 추가함

    SQL 관련 의존성 추가시 주의 할 점
        DataSource 관련 설정을 application.properties에 추가하지 않으면 제대로 작동하지 않음
        H2 와 같은 간편한 설정을 추가해두는게 좋아 보임

Github 로그인
    Github에서 이미 로그인된 계정 변경하기
        [1] git config --global user.name "$USER_NAME"
        [2] git config --global user.email "$USER_EMAIL"
        [3] Control Panel >> User Account >> Credential Manager >> Windows Credential >> Generic Credential
            이미 생성되어 있는 Github 관련 크레덴셜을 삭제해야 함
        [4] Clone > Push 를 통해서 해당 계정으로 다시 로그인 함
